## inertia remote rm

Remove a configured remote

### Synopsis

Remove a remote from Inertia's configuration file.

```
inertia remote rm [remote] [flags]
```

### Examples

```
inertia remote rm staging
```

### Options

```
  -h, --help   help for rm
```
